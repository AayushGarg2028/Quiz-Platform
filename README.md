# Quiz Platform

This repository contains a quiz platform application written in C. It includes an admin menu for managing questions and a student mode for playing the quiz.

## Getting Started

### Prerequisites
- A C compiler

### Running the Application
1. Compile the `main.c` file.
2. Run the executable.

### Admin Login
- Username: admin
- Password: admin123

### Questions File
The questions for the quiz must be placed in a file named `ques.txt` in the same directory as the executable.
